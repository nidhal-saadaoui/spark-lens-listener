ImpactEstimator

object ImpactEstimator

Shared helpers for computing EstimatedImpact values.

Speed assumptions are configurable via spark.sparklens.impact.* properties: spark.sparklens.impact.networkSpeedMbps (default 1024 — 1 GB/s cluster network) spark.sparklens.impact.diskSpeedMbps (default 200 — conservative SSD/HDD spill) spark.sparklens.impact.readSpeedMbps (default 512 — HDFS/S3 effective read)
